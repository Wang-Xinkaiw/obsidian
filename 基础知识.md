## 三、vector 和 queue 对比表

|   |   |   |
|---|---|---|
|特性|vector|queue|
|数据结构|动态数组|FIFO队列|
|内存布局|连续内存|依赖底层容器（默认deque）|
|随机访问|✅ 支持，O(1)|❌ 不支持|
|迭代器|✅ 支持|❌ 不支持|
|访问位置|任意位置|只能访问队首/队尾|
|添加位置|任意位置（末尾O(1)）|只能队尾（O(1)）|
|删除位置|任意位置（末尾O(1)）|只能队首（O(1)）|
|遍历方式|直接使用迭代器|需出队或复制|
|典型应用|数组操作、矩阵、需随机访问|BFS、任务调度、缓存|

### vector 注意事项

1. 迭代器失效：
    

```C++
vector<int> v = {1, 2, 3, 4, 5};auto it = v.begin() + 2;  // 指向3
v.insert(v.begin(), 0);    // 插入元素可能重新分配内存
// it 失效！不能继续使用
```

2. reserve vs resize：
    

```C++
vector<int> v;
v.reserve(100);    // 只分配内存，不创建元素// v[0] = 1;       // 错误！未初始化
v.resize(100);     // 分配内存并创建元素（默认值0）
v[0] = 1;          // 正确
```

3. 插入/删除性能：
    
    1. 末尾操作：O(1) 均摊
        
    2. 中间操作：O(n)
        
    3. 频繁在中间插入/删除考虑使用 `list` 或 `deque`
        
4. 二维vector的初始化：
    

```C++
// 正确初始化3×4矩阵
vector<vector<int>> matrix(3, vector<int>(4, 0));// 错误：vector<int>(3, 4) 是3个值为4的元素
// vector<vector<int>> wrong(3, 4);  
// 编译错误
```

### queue 注意事项

1. 不支持遍历：
    

```C++
queue<int> q;
q.push(1); q.push(2); q.push(3);// 错误：queue没有begin()/end()
// for(auto it = q.begin(); it != q.end(); ++it)
// 正确：复制队列或出队遍历
queue<int> temp = q;while(!temp.empty()) {
    cout << temp.front() << " ";
    temp.pop();}
```

2. 底层容器限制：
    

```C++
// queue默认使用deque
queue<int> q1;  // 底层为deque
// 可使用list，但不能用vector（无pop_front）
queue<int, list<int>> q2;  // √
// queue<int, vector<int>> q3;  
// × 编译错误
```

3. 线程安全：
    
    1. queue不是线程安全的
        
    2. 多线程访问需要加锁
        
    
    ```C++
    mutex mtx;
    queue<int> q;// 线程安全操作
    mtx.lock();
    q.push(data);
    mtx.unlock();
    ```
    

### 通用注意事项

1. 边界检查：
    

```C++
vector<int> v = {1, 2, 3};
// v[3] = 4;        // 未定义行为
// v.at(3) = 4;     // 抛出std::out_of_range异常

queue<int> q;
// q.front();      // 如果队列为空，未定义行为if(!q.empty()) 
{int val = q.front();  // 安全}
```

2. 类型匹配：
    

```C++
vector<int> v;
v.push_back(3.14);  // 3.14转换为int(3)，可能丢失精度
queue<double> q;
q.push(3);          // 3转换为double(3.0)，安全
```

3. 性能优化：
    

```C++
// vector预分配内存
vector<int> v;
v.reserve(1000);    // 避免多次重新分配

// 使用emplace避免拷贝
vector<pair<int, string>> vec;
vec.emplace_back(1, "hello");  // 直接构造
// 优于：vec.push_back(make_pair(1, "hello"));
```
