# 一般问题

## 重要成员函数

|   |   |   |
|---|---|---|
|函数|描述|时间复杂度|
|insert(value)|插入元素|O(log n)|
|erase(value)|删除所有值为value的元素|O(m + log n)|
|erase(iterator)|删除指定位置的元素|O(1) 平均|
|find(value)|查找元素|O(log n)|
|count(value)|统计元素出现次数|O(log n + m)|
|lower_bound(value)|返回第一个 ≥ value 的迭代器|O(log n)|
|upper_bound(value)|返回第一个 > value 的迭代器|O(log n)|
|equal_range(value)|返回相等元素的范围|O(log n)|

|   |   |   |
|---|---|---|
|函数|作用|是否修改原迭代器|
|std::prev(it)|获取前一个位置的迭代器|❌ 不修改|
|std::next(it)|获取后一个位置的迭代器|❌ 不修改|
|std::advance(it, n)|将迭代器前进/后退n位|✅ 修改|
|it-- 或 --it|迭代器自减|✅ 修改|

  

  

求最大子序列和（也称为最大子数组和）

```Python
def max_subarray_sum_kadane(nums):"""
    Kadane算法 - 时间复杂度 O(n)，空间复杂度 O(1)
    """if not nums:return 0
    
    max_sum = current_sum = nums[0]for i in range(1, len(nums)):# 当前元素 vs 当前元素+之前的和，取较大值
        current_sum = max(nums[i], current_sum + nums[i])# 更新全局最大值
        max_sum = max(max_sum, current_sum)return max_sum
```

  

### 构建线段树 (Build)

```Python
def build_tree(arr, tree, node, start, end):if start == end:# 叶子节点
        tree[node] = arr[start]else:
        mid = (start + end) // 2# 递归构建左右子树
        build_tree(arr, tree, 2node, start, mid)
        build_tree(arr, tree, 2node+1, mid+1, end)# 合并子节点信息
        tree[node] = tree[2*node] + tree[2*node+1]
```

### EOF处理方式不同

- `input()`: 遇到EOF（文件结束符）时，会抛出 `EOFError` 异常
    
- `sys.stdin.readline()`: 遇到EOF时，返回空字符串 `""`
    
- 最好用sys
